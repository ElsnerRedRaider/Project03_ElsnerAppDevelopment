# Elsner App Development Services

A website prototype for Elsner App Development Services, built as my final project for CMPA3301 (Introduction to Computing Applications) at Texas Tech University.

Elsner App Development Services is a business concept I intend to genuinely pursue, with HomeProof — an AI-powered home inventory app for insurance and legal use cases — as its flagship product. This repository is a two-page prototype of that future business's web presence, built to be an honest first step rather than a finished production site.

## Live Site

[View the live site](#https://elsnerredraider.github.io/Project03_ElsnerAppDevelopment/)

## Project Documentation

This project was managed using the framework from *Project Management for the Unofficial Project Manager*. The full planning documentation is in the [`docs/`](./docs) folder:

* [`scope.md`](./docs/scope.md) — the Project Scope Statement defining what this project is - and is not.
* [`plan.md`](./docs/plan.md) — a risk analysis built using the TAME framework and a Work Breakdown Structure.
* [`retrospective.md`](./docs/retrospective.md) — a reflection on the process once the project was complete.
